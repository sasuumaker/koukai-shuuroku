## 【Week 1】基礎編

---

### 第1回: Claude Codeの機能と用語を整理する

**目標**: よく聞く用語の違いを理解し、Skillsの仕組みを把握する

| 時間 | 内容 |
|------|------|
| 5分 | 4つの用語の紹介（Skills、標準コマンド、MCP、サブエージェント） |
| 3分 | 標準コマンド/MCP/サブエージェント（軽く説明） |
| 12分 | **Skillsの仕組み（メイン）** |
| 5分 | なぜSkillsが便利なのか |
| 5分 | まとめ・次回予告 |

**キーポイント**:
- Skills = Claudeが自動で認識して使ってくれる指示書（この講座のメイン）
- `.claude/skills/スキル名/SKILL.md` にファイルを置く
- YAMLフロントマターの `description` でいつ使うかを定義
- **段階的開示**: description → 本文 → 参照ファイルの順で効率的に読み込む

---

### 第2回: Skillsを作ってみよう

**目標**: 実際に手を動かして、最初のSkillを作成・実行する

| 時間 | 内容 |
|------|------|
| 2分 | 前回のおさらい（軽く） |
| 8分 | .claude/commands/ フォルダを作る |
| 10分 | 最初のSkillを作る（挨拶Skill） |
| 7分 | 引数を受け取るSkill（$ARGUMENTS） |
| 3分 | まとめ |

**ハンズオン**:
- フォルダ作成: `mkdir -p .claude/skills/hello-skill`
- `SKILL.md` を作成してClaudeに認識させる
- 実用的なツイート作成Skillも作成

---

### 第3回: 標準コマンドを使いこなす

**目標**: Claude Codeに最初から入っている便利なコマンドを覚える

| 時間 | 内容 |
|------|------|
| 15分 | /help /clear /context /compact の解説 |
| 10分 | その他の便利な標準コマンド |
| 5分 | まとめ |

**紹介するコマンド**:
- /help - ヘルプ表示
- /clear - コンテキストクリア
- /context - コンテキスト確認
- /compact - 手動圧縮

---

### 第4回: Skillsの書き方を深掘り

**目標**: より実用的なSkillsを書くためのテクニックを学ぶ

| 時間 | 内容 |
|------|------|
| 10分 | プロンプトの書き方のコツ |
| 10分 | 出力形式の指定方法 |
| 7分 | よくあるパターンと応用 |
| 3分 | まとめ |

**ポイント**:
- 役割を明確に指定する
- 出力形式（箇条書き、表、JSONなど）の指定
- 制約条件の書き方
